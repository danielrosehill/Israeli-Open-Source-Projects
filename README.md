# Israeli Open Source Projects

A curated index of interesting open-source Israeli projects — civic tech, scrapers, data pipelines, finance utilities, careers tools, real estate, smart home, media, and more.

This is a sibling to [Israeli AI Tools & Ecosystem](https://github.com/danielrosehill/Israeli-AI-Tools-And-Ecosystem). Where that repo focuses on AI-native projects, this one casts a wider net — including non-AI utilities and data sources that are useful inputs for AI projects, POCs, and analysis work.

*Last updated: 2026-04-20*
